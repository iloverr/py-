# 警醒图片替换说明

## 图片要求

当你通关全部10关后，会显示一张全屏警醒图片，配合红色快速闪烁效果。

## 如何替换图片

1. **准备图片文件**
   - 格式：JPG、PNG、GIF 等常见图片格式
   - 尺寸：建议 1920x1080 或更大，确保全屏显示效果
   - 内容：警醒、震撼、有冲击力的图片

2. **放置图片**
   - 将图片文件命名为 `alert.jpg`（或其他名称）
   - 放到以下目录：`d:\code_py\网页游戏\static\alert.jpg`

3. **修改代码**
   - 打开文件：`d:\code_py\网页游戏\templates\index.html`
   - 找到第 294 行左右，找到以下代码：
     ```html
     <img src="data:image/svg+xml,..." alt="警醒图片" id="alertImage">
     ```
   - 替换为：
     ```html
     <img src="/static/alert.jpg" alt="警醒图片" id="alertImage">
     ```

4. **重启游戏**
   - 刷新浏览器页面即可看到效果

## 效果说明

- **全屏显示**：图片会占满整个屏幕
- **红色闪烁**：红色半透明层会以 0.15秒 的间隔快速闪烁
- **强烈警醒**：配合视觉效果，达到强烈的警醒效果

## 当前状态

目前使用的是红色占位图（显示"警醒图片待添加"文字），等你准备好图片后可以替换。

## 示例代码

```html
<!-- 替换前 -->
<img src="data:image/svg+xml,..." alt="警醒图片" id="alertImage">

<!-- 替换后 -->
<img src="/static/alert.jpg" alt="警醒图片" id="alertImage">
```

## 注意事项

- 确保图片文件名和代码中的文件名一致
- 图片路径区分大小写
- 建议使用高对比度的图片，警醒效果更佳
